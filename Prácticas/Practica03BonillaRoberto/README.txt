Explicación como se compila la práctica: 

Es simple, solo abres la terminal en donde tienes guardado tu codigo con extencion .java y escribes "javac" + "nombre del archivo con terminación .java". Por ejemplo:

javac AnalizadorDeCadenas.java

En el caso de ejecutar un prorama que implemente una interfaz, estas deberan estar en la misma carpeta para que el programa compile y ejecute de manera correcta.

Para ejecutar es similar, solo omitimos la letra "c" y el .java. Por ejemplo:

java AnalizadorDeCadenas

Y en ambos casos damos ENTER.

Si tenemos errores, corregimos donde nos marque el compilador y repetimos el proceso hasta que la ejecucion sea como nosotros lo deseemos.
